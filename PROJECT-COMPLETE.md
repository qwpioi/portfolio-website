## 📝 使用说明



## 🎮 功能演示

### 正常浏览模式
1. 打开网站查看五行布局
2. 点击头像查看详细信息
3. 点击项目/文章/作品查看详情
4. 使用左右键或按钮切换上一个/下一个
5. 滚动页面查看返回顶部按钮

### 编辑模式
1. 按 `Ctrl+E` 或点击左上角头像 3 次
2. 进入管理页面编辑内容
3. 修改后点击"保存"或按 `Ctrl+S`
4. 点击"撤销"恢复修改前的状态
5. 点击"导出"备份数据为 JSON 文件
6. 按 `ESC` 或点击"退出"返回浏览模式
